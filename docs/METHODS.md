# Methods Directory

This document provides a comprehensive list of all the functions available through `is_one_one.py` used to prove that 1 is indeed 1. Community methods can live in `methods/*.py`; any function whose name starts with `is_one` is loaded automatically.

## Available Methods

#### `is_one()`
The only normal check in this entire repo that performs a direct comparison.
```python
from is_one_one import is_one

is_one() # Returns True
```

#### `is_one_unicode_distance()`
Verifies that the Unicode distance between 'a' and 'b' is exactly 1.
```python
from is_one_one import is_one_unicode_distance

is_one_unicode_distance() # Returns True
```

#### `is_one_using_time_travel()`
Uses the current year and some useless math to always return True.
```python
from is_one_one import is_one_using_time_travel

is_one_using_time_travel() # Returns True
```

#### `is_one_using_interdimensional_tax_fraud()`
Combines cosmic alignment, government surveillance IDs, and quantum probability.
```python
from is_one_one import is_one_using_interdimensional_tax_fraud

is_one_using_interdimensional_tax_fraud() # Returns True
```

#### `is_one_using_binary()`
Parses the binary string '1' using base-2 logic to verify it equals 1.
```python
from is_one_one import is_one_using_binary

is_one_using_binary() # Returns True
```

#### `is_one_under_extreme_pressure()`
Hides the number 1 inside 50 layers of dictionaries and manually digs it back out.
```python
from is_one_one import is_one_under_extreme_pressure

is_one_under_extreme_pressure() # Returns True
```

#### `is_one_using_roman_numerals()`
Calculates the value of the Roman numeral "I".
```python

is_one_using_roman_numerals() # Returns True
```

#### `is_one_just_to_be_sure()`
The ultimate recursive confirmation that runs all other checks to be absolutely sure.
```python

is_one_just_to_be_sure() # Returns True
```
